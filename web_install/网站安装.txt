重命名/mnt/hda3/目录下www  mysql两个目录

www根目录下创建web_install连接/mnt/sda3/web_install
访问目录为http://192.168.10.1/web_install

更新软件源
opkg update

安装python-pyopenssl
opkg install python-pyopenssl
opkg install python-openssl
opkg install kmod-ath9k --force-depends

安装PHP、MYSQL等程序
sh /mnt/sda3/PHP7install/web_install.sh
卸载
sh /mnt/sda3/PHP7install/web_remove.sh

更新nginx配置文件  更正www mysql目录
------------------------------------

安装支持ssl的nginx
opkg install /mnt/sda3/web_install/nginx_1.10.3-2_x86_64.ipk

nginx配置测试
nginx -t

如仅安装程序需初始化
mysql_install_db --force
/etc/init.d/mysqld enable
/etc/init.d/mysqld start
/usr/bin/mysqladmin -u root password 456789xx
groupadd www
useradd -g www www
chown -R www:www /www
/etc/init.d/php5-fastcgi enable
/etc/init.d/php5-fastcgi start
/etc/init.d/nginx enable
/etc/init.d/nginx start

你可以在终端使用 www 命令来控制网站
www start   启动服务器
www stop    关闭服务器
www restart 重启服务器

上传文件到网站时记得设置所有者为: www 用户组: www
你可以使用 /etc/init.d/wwwset start 来快捷设置
你可以使用phpmyadmin 来管理你的数据库 增与删
数据库用户名：root 密码：456789xx
